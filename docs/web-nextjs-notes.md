# Next.js boilerplate notes (`apps/web`)

The `**apps/web**` directory is the Next.js app (from `create-next-app`). There is **no** `README.md` inside that folder—**only** the [root README](../README.md) lives outside `docs/`. **Lumina-specific** setup (env, workspaces, Supabase) is in the root README handbook and [Getting started](getting-started.md).

## Default dev command (from `apps/web` only)

```bash
npm run d
```

Open [http://localhost:3000](http://localhost:3000). From the **repo root**, prefer `npm run dev:web`.
